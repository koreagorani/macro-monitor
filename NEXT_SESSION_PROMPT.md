# 다음 세션 시작 프롬프트

이 프로젝트는 주간 매크로 모니터링 및 포트폴리오 위험 점검 시스템이다.

이번 세션의 역할은 **GitHub 저장소와 하네스 문서 구조 생성**이다.

먼저 아래 문서만 읽어라.

- `AGENTS.md`
- `docs/ARCHITECTURE.md`
- `docs/ROADMAP.md`
- `docs/HANDOFF.md`

필요할 때만 `docs/REQUIREMENTS.md`를 추가로 읽어라.
위 목록 외 문서는 현재 작업에 필요하지 않으면 읽지 마라.

목표:
1. GitHub 저장소의 실제 폴더와 파일 구조를 만든다.
2. 문서 파일을 적절한 위치에 배치한다.
3. config 파일의 빈 초안을 만든다.
4. README에 프로젝트 목적과 첫 실행 단계를 적는다.
5. 작업 후 `docs/HANDOFF.md`를 갱신한다.

아직 데이터 수집 코드나 위험 계산 코드는 구현하지 마라.
