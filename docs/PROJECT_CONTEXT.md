# Project Context

## 목적

주간 매크로 정보 확인과 포트폴리오 점검을 자동화한다.

최종 흐름:

```text
데이터 수집
→ 지표 상태 판정
→ 영역별 위험 계산
→ 포트폴리오 테마별 취약도 계산
→ AI 해석
→ 주간 보고서 생성
→ Notion 저장
→ 중요 이벤트 경고
→ 추후 Telegram 전송
```

## 핵심 원칙

- 자동매매는 하지 않는다.
- 취약도는 기대수익률이나 매도 신호가 아니다.
- 취약도는 현재 매크로 환경에 대한 노출 정도다.
- 숫자 판정과 AI 해석을 분리한다.
- 지표와 포트폴리오가 바뀌어도 핵심 알고리즘이 유지되어야 한다.
- 공식 데이터와 주요 언론을 우선한다.
- 보고서는 5분 안에 핵심을 파악할 수 있어야 한다.

## 사용자 환경

- 군대 컴퓨터
- 로컬 설치와 터미널 사용 제한 가능
- 브라우저 중심 개발
- GitHub, Vercel, StackBlitz 우선
- 낮은 개발 숙련도를 고려한 단계적 진행
- 유료 서비스 사용 전 비용 확인

## 장기 확장

- 지표 확대
- AI API 자동 분석
- Notion 자동 저장
- Telegram 경고
- 과거 판단 평가와 임계값 튜닝
