# Requirements

## MVP 지표

- 미국 2년물 국채금리
- 근원 PCE
- WTI 유가
- 원·달러 환율
- 비트코인
- S&P 500

## 향후 추가

자주 확인:
- 미국 10년물
- 연준 금리 기대
- 비농업고용
- 금

저빈도:
- TGA
- M2
- 미국 국채 발행
- 미국 재정정책

## 실행 주기

- 주 1회 보고서
- 중요 이벤트 별도 경고

## 비교 기준

시장가격형:
- 전주 대비
- 4주 누적
- 향후 최근 3년 주간 변화 백분위

정기발표형:
- 이전 발표치
- 시장 예상치
- 3개월 추세

## 보고서

- 5분 이내에 핵심 판단 가능
- 전체 위험 단계
- 영역별 위험
- 지표 코멘트는 특이사항이 있을 때만
- 포트폴리오 취약도는 테마 단위
- 기본적으로 상위 3개 취약 테마 표시
- 헷지는 필요성이 있을 때만 1~2개 제안
- 다음 주 조건과 판단 기록
- 세부 출력 규칙은 `docs/REPORT_SPEC.md`를 따른다

## 특이사항 조건

- 과거 분포상 이례적 움직임
- 전쟁·관세·정책·공급 충격
- 다른 지표와 상충
- 예상치와 유의미한 차이
- 추세 전환 또는 주요 기준선 돌파
- 데이터 오류·휴장·발표 지연

## 자동화 범위

AI 없이:
- 데이터 수집
- 변화율
- 임계값
- 영역 점수
- 테마 취약도
- 표 생성
- Notion 저장

AI:
- 사건 요약
- 상충 신호 설명
- 일시적/지속적 변화 구분
- 포트폴리오 영향
- 헷지 설명
- 자연어 보고서
