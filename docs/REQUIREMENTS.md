# Requirements

## MVP 지표

- 미국 2년물 국채금리
- 근원 PCE
- WTI 유가
- 원·달러 환율
- 비트코인
- S&P 500

## 향후 추가

자주 확인:
- 미국 10년물
- 연준 금리 기대
- 비농업고용
- 금

저빈도:
- TGA
- M2
- 미국 국채 발행
- 미국 재정정책

## 실행 주기

- 주 1회 보고서
- 중요 이벤트 별도 경고

## 비교 기준

시장가격형:
- 기준일 현재값
- 전주 대비 변화율
- 4주 누적 변화율
- 향후 최근 3년 주간 변화 백분위
- 필요한 경우 최근 흐름 또는 추세 판정

정기발표형:
- 이전 발표치
- 시장 예상치
- 3개월 추세

근원 PCE:
- 월간 발표형 지표로 처리한다.
- 이전 발표치와 3개월 추세를 기본 비교 기준으로 사용한다.
- 시장 예상치는 신뢰할 수 있는 출처가 확보된 경우에만 사용한다.

## 데이터 조회 및 보관

- 실행 시점에 필요한 기간의 데이터만 API로 조회한다.
- 원시 시계열은 GitHub Actions의 메모리 또는 임시 파일에서만 처리한다.
- 계산이 끝난 뒤 원시 시계열은 폐기한다.
- 비트코인과 S&P 500의 전체 원시 시계열을 공개 저장소에 장기간 저장하거나 커밋하지 않는다.
- 공개 저장소에는 데이터 수집 코드, 설정, 임계값만 보관한다.
- 현재값과 변화율이 포함된 최종 보고서는 저장소에 커밋하지 않고 개인 Notion 데이터베이스로 전송한다.
- Telegram 연동 후에는 핵심 요약과 경고만 전송한다.

## 로그 및 출처

- GitHub Actions 로그에 원시 시계열 전체를 출력하지 않는다.
- 오류 로그에 API 응답 본문 전체를 출력하지 않는다.
- 보고서에는 각 지표의 원출처와 기준일을 표시한다.

## 보고서

- 5분 이내에 핵심 판단 가능
- 전체 위험 단계
- 영역별 위험
- 지표 코멘트는 특이사항이 있을 때만
- 포트폴리오 취약도는 테마 단위
- 기본적으로 상위 3개 취약 테마 표시
- 헷지는 필요성이 있을 때만 1~2개 제안
- 다음 주 조건과 판단 기록
- 세부 출력 규칙은 `docs/REPORT_SPEC.md`를 따른다

## 특이사항 조건

- 과거 분포상 이례적 움직임
- 전쟁·관세·정책·공급 충격
- 다른 지표와 상충
- 예상치와 유의미한 차이
- 추세 전환 또는 주요 기준선 돌파
- 데이터 오류·휴장·발표 지연

## 자동화 범위

AI 없이:
- 데이터 수집
- 변화율
- 임계값
- 영역 점수
- 테마 취약도
- 표 생성
- Notion 저장

AI:
- 사건 요약
- 상충 신호 설명
- 일시적/지속적 변화 구분
- 포트폴리오 영향
- 헷지 설명
- 자연어 보고서
